In this article I want share the keyboard shortcuts that I use on a daily basis, in various environments.

Advantages of using shortcuts:
- Shave off a few seconds or milliseconds, hundreds of times over. The time savings really add up.
- Make you look smarter in screen sharing sessions, live demos and pairing.
- They're just fun to use!

## iTerm2

![](/images/tools/iterm2.png?size=icon-small)

| Shortcut                                       | Action                   | 
| :-                                             | :-                       |
| ( <kbd>↑</kbd> / <kbd>↓</kbd> )                | Previous / Next history  |
| <kbd>⌃</kbd> + R                               | Search history           |
| <kbd>⌃</kbd> + C                               | Stop current process     |
| <kbd>⌃</kbd> + ( <kbd>A</kbd> / <kbd>E</kbd> ) | Skip to line start / end |
| <kbd>⌃</kbd> + <kbd>W</kbd>                    | Delete previous word     |
| <kbd>⌃</kbd> + D                               | Delete next character    |
| <kbd>⌘</kbd> + T                               | Open new tab             |
| <kbd>⌘</kbd> + <kbd>N</kbd>                    | Open new window          |

Some additional convenience shortcuts can be added by [configuring iTerm2 key mappings](https://mariusschulz.com/blog/keyboard-shortcuts-for-jumping-and-deleting-in-iterm2), as described in Marius Schulz's excellent article.

| Shortcut                                       | Action                    | 
| :-                                             | :-                        |
| <kbd>⌥</kbd> + ( <kbd>←</kbd> / <kbd>→</kbd> ) | Skip by word left / right |
| <kbd>⌘</kbd> + ( <kbd>←</kbd> / <kbd>→</kbd> ) | Skip to line start / end  |
| <kbd>⌥</kbd> + <kbd>⌫</kbd>                    | Delete previous word      |
| <kbd>⌘</kbd> + <kbd>⌫</kbd>                    | Delete line               |

## Fork

![](/images/tools/fork.png?size=icon-small)

| Shortcut                                                      | Action                                                          | 
| :-                                                            | :-                                                              |
| <kbd>⌘</kbd> + <kbd>C</kbd>                                   | Copy full file name of the selected file. Including path.       |
