In this article I want share the keyboard shortcuts that I use on a daily basis, in various environments.

Advantages of using shortcuts:
- Shave off a few seconds or milliseconds, hundreds of times over. The time savings really add up.
- Make you look smarter in screen sharing sessions, live demos and pairing.
- They're just fun to use!

## Visual Studio Code

![](/images/tools/vs-code.png?size=icon-small)

| Shortcut                                           | Action                               | 
| :-                                                 | :-                                   |
| `⌘ Command + ⇧ Shift + ⌥ Option + ↑ Up / ↓ Down`   | Rectangular selection                |
| `⇧ Shift + ⌥ Option + 🖱️ Drag`                     | Rectangular selection (mouse)        |
| `⌘ Command + A`                                    | Select all                           |
| `⌘ Command + L`                                    | Select line                          |
| `⌃ Control + A  / E`                               | Skip to line start / end             |
| `⌥ Option + ← Left / → Right`                      | Skip by word left / right            |
| `⌥ Option + ⇧ Shift + ← Left / → Right`            | Select by word left / right          |
| `⌥ Option + ↑ Up / ↓ Down`                         | Move line up / down                  |
| `⌥ Option + ⇧ Shift + ↑ Up / ↓ Down`               | Duplicate line up / down             |
| `⌘ Command + ⇧ Shift + [ / ]`                      | Switch tab left / right              |
| `⌃ Control + ⇥ Tab / ⇧ Shift + ⇤ Tab`              | Switch tab most / least recent       |
| `⌘ Command + [ / ]`                                | Indent / outdent                     |
| `⌘ Command + /`                                    | Toggle comment                       |
| `⌘ Command + W`                                    | Close current tab                    |
| `⌘ Command + K, W`                                 | Close all tabs. Keep window open.    |
| `⌘ Command + ⇧ Shift + W`                          | Close current window                 |
| `⌘ Command + P`                                    | Quick find file                      |
| `⌘ Command + ⇧ Shift + P`                          | Quick find action                    |
| `⌘ Command + T`                                    | Quick find symbol                    |
| `⌘ Command + N`                                    | Open new tab                         |
| `⌘ Command + ⇧ Shift + N`                          | Open new window                      |
| `⌘ Command + ⇧ Shift + E`                          | Focus on file browser                |
| `⌘ Command + F2`                                   | Select all instances of current word |

## Fork

![](/images/tools/fork.png?size=icon-small)

| Shortcut                                                     | Action                                              | 
| :-                                                           | :-                                                  |
| `⌘ Command + C` (on a file in stages or unstaged)            | Copy full file name. Including path.                |
| `⌘ Command + ⇧ Shift + O` (on a file in stages or unstaged)  | Open the file. Using default editor.                |
| `⌘ Command + ⇧ Shift + C`                                    | Focus on the commit message input.                  |
| `↓ Down` (on the commit message input)                       | Show recent commit messages. Click to re-use.       |
| `⌘ Command + ⇧ Shift + [ / ]`                                | Switch tab left / right                             |
| `⌘ Command + T`                                              | Open new tab                                        |
| `⌘ Command + N`                                              | Open new window                                     |
