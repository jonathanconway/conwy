TailwindJS is a Javascript/Typescript library that exposes [TailwindCSS](https://tailwindcss.com) classes as Javascript objects.


TailwindJS exposes:

- **All TailwindCSS utilities** as simple constants.
  - Example: `h_10` => `h-10`
- **All TailwindCSS modifiers** as functions.
  - Example: `dark(bg_stone_900)` => `dark:bg-stone-900`

Additionally TailwindJS provides:

- **Arbitraries**
  - Example: `h_arb("30rem")`
- **Helpers**
  - Example: negative values (`neg`)
  - Example: class name joiner (`classNames`)
